# Telegram Subscription Bot

A complete button-based subscription bot for private Telegram channels.

## Features

- ✅ 100% Button-based interface
- ✅ Multiple payment methods (UPI, Bank, Crypto, etc.)
- ✅ Admin control panel
- ✅ Automatic subscription management
- ✅ Referral system with commissions
- ✅ Payment verification system
- ✅ Database backups
- ✅ Broadcast messages
- ✅ Expiry reminders

## Setup Instructions

### 1. Prerequisites
- Python 3.8 or higher
- Telegram account
- Private Telegram channel

### 2. Installation

```bash
# Clone or create project directory
mkdir subscription-bot
cd subscription-bot

# Install dependencies
pip install -r requirements.txt

# Configure environment
cp .env.example .env
# Edit .env with your details
# telegram-vip-subscription-bot
